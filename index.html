Required Tools

Publishing a repository with GitHub pages is very simple, and requirements are minimal. For resources about these tools, see the "Resources" section.

The required tools:

An internet connection to GitHub's website https://github.com/ A public repository with a README.md file Some knowledge of the language Markdown

Other recommended tools:

Knowledge of HTML, CSS, and JavaScript A version control tool (such as Git or SVN) if working collaboratively
